# Selenium + RestAssured Automation Framework

Framework de automatización para pruebas **Web** y **API**, desarrollado en **Java** utilizando **Selenium WebDriver**, **RestAssured**, **JUnit 5**, arquitectura **Page Object Model (POM)** y reportes con **Extent Reports**.

Este proyecto combina automatización UI (Web) y API bajo una estructura modular, escalable y fácil de mantener.
